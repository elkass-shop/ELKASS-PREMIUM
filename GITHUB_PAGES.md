# GitHub Pages

Wgraj zawartość paczki bez dodatkowego katalogu nadrzędnego.

Projekt automatycznie wykrywa nazwę repozytorium na domenie `*.github.io` i ustawia bazę adresów. Dzięki temu `/assets`, `/app`, panel oraz podgląd motywu działają również pod adresem typu `/ELKASS-PREMIUM/`.

Po aktualizacji wykonaj twarde odświeżenie: Ctrl+F5 lub wyczyść cache witryny.

# Publikacja ELKASS na GitHub Pages

1. Wgraj **zawartość** tego folderu do głównego katalogu repozytorium.
2. W GitHub: Settings → Pages → Source: **GitHub Actions**.
3. Otwórz Actions i poczekaj na zielony status workflow „Deploy GitHub Pages”.
4. Strona będzie pod adresem `https://LOGIN.github.io/NAZWA-REPO/`.

Repo ma poprawione ścieżki względne i działa także w podkatalogu projektu.
Panel: `https://LOGIN.github.io/NAZWA-REPO/admin/`.
